class AdvertisementData¶

Serializes Adversitement packet data.

Attention

1. Adversitement data has a maximum of only 31 bytes, no more fields can be added after that.

Attention

2. This serializer doesn’t implement all fields of GAP adversitement, only those that are really needed here are implemented.
