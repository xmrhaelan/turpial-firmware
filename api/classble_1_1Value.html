void addPart(std::uint8_t *value, std::size_t len)¶

Add a part of the write from a data buffer.

Attention

1. This method is used internally by Characteristic for partial writes, don’t use this method outisde of Characteristic.

Attention

2. After you are complete adding parts you must call commit() or cancel() to set the value or to cancel the write respectively.

Parameters

  • value: data buffer

  • len: data buffer length
