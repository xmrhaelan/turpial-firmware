class CharacteristicCallback¶

Callback handler class, this is intended to be used by characteristics to know when a value has been written/readed.

Subclassed by ble_preferences::APPasswordCallback, ble_preferences::APSSIDCallback, ble_preferences::STAPasswordCallback, ble_preferences::STASSIDCallback, ble_preferences::WiFiModeCallback
